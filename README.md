## :dart: Starting ##

```bash
# Clone this project
$ git clone https://github.com/tmchuynh/real-estate-crm

# Access
$ cd real-estate-crm

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
